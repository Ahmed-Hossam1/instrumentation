# Instrumentation App

The **Instrumentation App** is a modern web application designed to serve as a comprehensive dashboard and data tracking platform. Built with a robust full-stack architecture, it provides an intuitive interface for data visualization, monitoring, and seamless user experiences backed by a secure database.

## ✨ Key Features

- **Interactive Dashboards:** Real-time data visualization using highly customizable and responsive charts.
- **Secure Authentication:** Complete user session management and authentication flow powered by Supabase.
- **Beautiful UI Elements:** Modern, accessible, and clean user interface with fluid animations and transitions.
- **Responsive Design:** Fully responsive layout that adapts to desktops, tablets, and mobile devices effortlessly.
- **Real-time Data Sync:** Capability to handle backend data syncing and realtime updates via a managed PostgreSQL database.

## 🧑‍💻 What Users Can Do

By interacting with the application, users are able to:
- **Monitor Metrics:** View dynamically updated, highly readable charts reflecting real-time system or application data.
- **Account Management:** Quickly create an account, securely sign in, and maintain an active authenticated session.
- **Analyze Data Trends:** Check historical data and patterns using the interactive graphs.
- **Customizable Interface:** Navigate smoothly between secure pages and interact with a highly responsive, animated UI.
- **Receive Instant Feedback:** Get immediate visual updates on their actions (success, warnings, errors) via pop-up notifications.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the project by modifying `src/app/page.tsx` (or `app/page.tsx`). The page auto-updates as you edit the file.
